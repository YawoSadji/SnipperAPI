Welcome to Snippets API

Please check out the examples and read the instructions to better understand how to access the right informations.

  1. To register, please send a post request to the snippets/register endpoint with a body containing a json formatted email and password.
  2. To login, please send a post request to the snippets/login endpoint with a body containing your json formatted email and password.
  3. To get all your snippets, please send a get request to the snippets endpoint
  4. To get a specific snippet, please send a get request to the snippets/id endpoint
  5. To add a snippet, please send a post request to the snippets/add
